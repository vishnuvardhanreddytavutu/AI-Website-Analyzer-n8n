## 🚀 Overview  
This project is an AI-powered system built using **n8n** that analyzes websites and generates business insights along with competitor identification. It shows how automation and AI can turn raw website data into meaningful information instantly.  

---

## 🧠 Features  
- AI-powered website analysis  
- Automated insight generation  
- Competitor detection  
- Trigger-based workflow using URL input  
- Structured output generation  
- Scalable no-code/low-code design  

---

## 🛠️ Tech Stack  
- n8n  
- AI (GPT-based processing)  
- REST APIs  
- JSON  

---

## 🔄 How It Works  
1. User submits a website URL  
2. n8n workflow is triggered  
3. Website data is extracted and processed  
4. AI analyzes the content  
5. Insights and competitors are generated  
6. Output is returned in structured format
